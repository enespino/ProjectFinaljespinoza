Introducción
El objetivo de esta practica es asentar los conocimientos que se han obtenido durante el curso.
Para ello vamos a crear un clasificador de tipos de carnes que se utiliza en la industria real. Es un ejemplo
simplificado, el modelo real utiliza imágenes de mayor resolución y el conjunto de entrenamiento es mucho
mayor (requiriendo días para su entrenamiento). Pero para practicar, con esta pequeña muestra nos sirve.
